# Theme colours and editing

Theme color object is build as below

```typescript
defaultTheme = {
    colors:{
      primary:{
        light?: string;
        main?: string;
        dark?: string;
      },
      secondary:{
        light?: string;
        main?: string;
        dark?: string;
      },
      tertiary:{
        light?: string;
        main?: string;
        dark?: string;
      },
      info:{
        light?: string;
        main?: string;
        dark?: string;
      },
      success:{
        light?: string;
        main?: string;
        dark?: string;
      },
      error:{
        light?: string;
        main?: string;
        dark?: string;
      },
      warning:{
        light?: string;
        main?: string;
        dark?: string;
      },
      singleTone:{
        white?: string;
        black?: string;
        lightText?: string;
        darkText?: string;
      },
      grey:{
        50?: string;
        100?: string;
        200?: string;
        300?: string;
        400?: string;
        500?: string;
        600?: string;
        700?: string;
        800?: string;
        900?: string;
      }
    }
  }
```

To edit the colours import the ComponentsThemeProvider wrap your app with it and add your new theme as a props,
the new colours will be merged with the default theme only replacing the colours you've defined in your new theme

In the below example the primary colours have been edited

```jsx
function App() {
  const theme = {
    colors: {
      primary: {
        light: 'lightred',
        main: 'red',
        dark: 'darkred',
      },
    },
  };

  return (
    <div>
      <ComponentsThemeProvider theme="theme">
        ... app components
      </ComponentsThemeProvider>
    </div>
  );
}
```
